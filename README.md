# 📌 Purpose: Multilingual Quran & Hadith API for Developers

This repository serves **developers worldwide** who need **authentic, multilingual Islamic data** from trusted official platforms.

It provides unified access to:

- **IslamHouse.com** → Books, fatwas, articles, audio, video, classifications in 130+ languages
- **QuranEnc.com** → Quran translations & tafsir in 70+ languages
- **HadeethEnc.com** → Verified hadith translations & explanations in 60+ languages

All resources are official, reliable, and freely accessible.

---

### Reason for Existence

Many developers struggle to find **official Islamic APIs with reliable multilingual support**.
This hub eliminates scraping, unofficial sources, and fragmentation by offering a single, trusted reference point.
